# Display fractals

## Requirements

- [GLFW3](https://www.glfw.org/download.html)
- [c-fractals](../c-fractals)

## Keyboard shortcuts

| Shortcut           | Description            |
| ------------------ |:----------------------:|
| **q**              | Quit                   |
| **r**              | Reset the view         |
| **W,A,S,D**        | Move around the plane  |
| **scroll up/down** | zoom in/out            |
| **left/right**     | rotate the fractal     |
| **1**              | Default                |
| **2**              | Threaded               |
| **3**              | AVX                    |
| **4**              | AVX Threaded           |
